회원가입. ID, PW, 개인정보(실명, 닉네임, 성별, 전화번호, 이메일)를 전달받는다.

`POST` `http://localhost:8080/auth/join`

### URI Parameter

### Request Body

DB `member`, `memberInformation`

```json
{

    "name": "홍길동",
    "member": {
        "username": "vcho1958",
        "password": "dfsjbnjodsfno"
    },
    "nickname": "참새",
    "gender": "male",
    "phone": "01065647895",
    "email": "gasdf@asd.com"
}
```





### Response

| Status code | Type     | Description |
| ----------- | -------- | ----------- |
| 200 OK      | JoinForm | id포함 정보 |
| 500         |          | 서버 오류   |
